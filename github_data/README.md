# GitHub Actions用公開データ

このフォルダだけを、朝のLINE通知がGitHub Actions上で参照する。

## 入れてよい情報

- SNSで公開してよい家づくり進捗の要約
- 公開してよいAI活用テーマ
- 商品名、公開用レビュー、公開用Amazonリンク
- 個人を特定できない投稿分析結果

## 入れてはいけない情報

- 氏名、住所、電話番号、メールアドレス
- 土地の所在地、契約書、ローン番号、口座・支払情報
- LINEトークン、Channel Secret、userId
- Amazon APIキー、その他の認証情報
- `.env`、写真、見積書原本

実データを更新するときは、ObsidianのHouse_DBを丸ごとコピーせず、公開可能な要約だけを手入力する。
